Settings

This works even if the translation list fails to load (fallback options are built in).
Curated mode picks from the local reference list (English references) and fetches text per translation. API mode uses bible-api.com random.
Note: some translations do not support scoped random (NT/OT/etc). This app auto-falls back.
When On: Translation 1 -> 2 -> 3 (if set), then a NEW verse starts again at Translation 1.
When On: top buttons and hints hide after 5 seconds of no taps. Tap to show again. Also hides the translation line below the verse.
iOS requires one tap before audio can start. First tap will start music if On.
This version loads background images and MP3 files from the same folder as this HTML file.
